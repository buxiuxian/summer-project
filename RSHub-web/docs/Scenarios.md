# Supporting Scenarios

## Soil scenario 
✅ **NMM3D-VIE-DDA**: wave approach based on Volume Integral Equation and Discrete Dipole Approximation for layered soils

⬜ **NMM3D-SIE**:  wave approach based on Surface Integral Equation (upcoming)

## Snow scenario 
✅ **DMRT-QMS**: Dense Media Radiative Transfer Model- the Quasi-Crystalline Approximation of Mie scattering of densely packed Sticky spheres

✅ **DMRT-BIC**: Dense Media Radiative Transfer Model- the bicontinuous dense media radiative transfer

⬜ Full wave approach based on bicontinuous media and the half-space Green’s function (upcoming)

## Vegetation scenario 
✅ **VPRT**: Vegetation Passive Radiative Transfer model

⬜ **VART**: Vegetation Active Radiative Transfer model (upcoming)

⬜ Wave approach currently in development (upcoming)
