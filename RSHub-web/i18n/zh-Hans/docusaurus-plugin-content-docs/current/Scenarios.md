# 支持的场景

## 土壤场景
✅ **NMM3D-VIE-DDA**：基于体积积分方程和离散偶极子近似的分层土壤波动方法

⬜ **NMM3D-SIE**：基于表面积分方程的波动方法（即将上线）

## 积雪场景
✅ **DMRT-QMS**：致密介质辐射传输模型——密集粘结球体的Mie散射准晶体近似

✅ **DMRT-BIC**：致密介质辐射传输模型——双连续介质辐射传输

⬜ 基于双连续介质和半空间格林函数的全波方法（即将上线）

## 植被场景
✅ **VPRT**：植被被动辐射传输模型

⬜ **VART**：植被有源辐射传输模型（即将上线）

⬜ 波动方法开发中（即将上线）
